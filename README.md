# Hall Laser Steel Landing Page

Este é um projeto React profissional desenvolvido para a Hall Laser Steel, com foco em estética industrial tecnológica premium.

## ✨ Funcionalidades
- **Hero Section Cinemática**: Animação de corte laser controlada pelo scroll do mouse (Canvas).
- **Seção de Serviços**: Blocos alternados com detalhes técnicos e produtos.
- **Carrossel de Produtos**: Scroll infinito de projetos desenvolvidos.
- **Responsividade Total**: Otimizado para Mobile e Desktop.
- **Design Premium**: Cores institucionais (#0B1F3A, #0E5EFF) com efeitos de glow e glassmorphism.

## 🚀 Como Executar

### 1. Instalar Dependências
```bash
npm install
```

### 2. Rodar em Desenvolvimento
```bash
npm run dev
```

### 3. Gerar Build de Produção
```bash
npm run build
```

## 📂 Estrutura de Pastas
- `public/imagens/`: Todas as imagens organizadas e renomeadas para o projeto.
- `src/components/`: Componentes modulares (Hero, Services, Contact, etc.).
- `src/App.jsx`: Estrutura principal e navegação.

---
Desenvolvido com React, Vite e Framer Motion.
